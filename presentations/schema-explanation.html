├─ 📚 Course ┬ Created by instructor (FK: instructorId)

│ └─ 📋 Enrollment ┬ Junction: User ↔ Course

├─ 🏆 Certificate Awarded on course completion
└─ ⭐ Review Rating & feedback per enrollment

Fields

Constraints
@@unique([enrollmentId, lessonId])
One progress record per enrollment per lesson
  • progress on Enrollment is a computed % — updated whenever a lesson is marked complete
  • This design allows tracking completion at the lesson level while aggregating up to course level

QuizAttempt Student submission

Fields
idUUID
scoreInt
maxScoreInt
answersJSON
passedBoolean
startedAtDateTime
completedAtDateTime?
userIdString
quizIdString
Constraints
@@index([userId, quizId])
Multiple attempts allowed — limit set by quiz.attemptsAllowed (null = unlimited)
  • options stores the choices as JSON (["A", "B", "C", "D"]), correctAnswer stores the correct one
  • Quiz grading is done server-side: score = sum(points where answer === correctAnswer)
  • attemptsAllowed is set by the instructor — null means unlimited attempts, 1 = one try only
